Cascading Style Sheets

Introducing CSS

CSS works by allowing you to associate rules with the elements that appear in a web page. These rules govern how the content of those elements should be rendered. Figure 7 - 1 shows you an example of a CSS rule, which is made up of two parts:
  1. The selector , which indicates which element or elements the declaration applies to (if it applies to more than one element, you can have a comma - separated list of several elements)
  2. The declaration , which sets out how the elements referred to in the selector should be styled
The declaration is also split into two parts, separated by a colon:
  1. A property , which is the property of the selected element(s) that you want to affect, in this case the width property.
  2. A value , which is a specification for this property; in this case it is that the table cells should be 36 pixels wide.
